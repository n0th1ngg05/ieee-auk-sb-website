IEEE Amity University Kolkata Student Branch Website

Create a modern, professional, multi-page website for the IEEE Amity University Kolkata Student Branch (AUK SB).

The website must look like a technical academic organization website, similar to IEEE, MIT labs, or university research group sites.

Design inspiration:
A modular homepage layout similar to the Essex Summer School website, with:

• large hero banner
• modular grid content blocks
• statistics counters
• academic sections
• professional typography
• strong structure

The website must be fully responsive and structured for future database integration.

Use only:

HTML
CSS
Vanilla JavaScript

Do not use frameworks like React or Angular.

PROJECT DIRECTORY STRUCTURE

Use exactly the following structure:

root

├── index.html
├── about.html

videos
